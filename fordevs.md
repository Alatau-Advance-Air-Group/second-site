# fordevs.md — AAAG Demo Day Landing Page

## Цель проекта

Одностраничный лендинг для мероприятия AAAG.kz (19 мая 2026, Алматы):
демонстрационные полёты авиатакси и доставка дронами. Гости регистрируются через форму,
заявки уходят на email через Formspree.

## Структура файлов

```
site_for_invitations/
├── index.html   — весь сайт (HTML + CSS + JS, один файл)
├── todo.md
└── fordevs.md
```

## Секции страницы

1. **Навигация** — логотип AAAG.kz, live-badge, кнопка «Регистрация»
2. **Hero** — заголовок, описание, мета (дата/место/формат), обратный отсчёт, радар-анимация на canvas
3. **О мероприятии** — текст + статы (01 / Live / B2B)
4. **Программа** — 4 карточки: демо-полёты, доставка дронами, тех-презентация, нетворкинг
5. **Регистрация** — форма (Имя, Фамилия, Должность/Компания) → Formspree
6. **Футер** — логотип, дата/место, ссылка на aaag.kz

## Технологии

- Чистый HTML/CSS/JS, без фреймворков
- Google Fonts: Orbitron (заголовки), IBM Plex Mono (mono), IBM Plex Sans (тело)
- Canvas API — анимированный радар в hero-секции
- Formspree — отправка формы на email без бэкенда
- Intersection Observer API — scroll-reveal анимации

## Настройка Formspree (ОБЯЗАТЕЛЬНО перед деплоем)

1. Зайти на [formspree.io](https://formspree.io) и создать аккаунт
2. Создать новую форму, указать email-адрес получателя
3. Скопировать ID формы (вида `xabc1234`)
4. В `index.html` найти строку:
   ```html
   <form id="reg-form" action="https://formspree.io/f/YOUR_FORM_ID"
   ```
   и заменить `YOUR_FORM_ID` на реальный ID

## Поля формы

| Поле | name-атрибут | Обязательное |
|------|-------------|--------------|
